Welcome to the Vizrt APT repository

To use this repository, do: 

# curl -s http://apt.vizrt.com/archive.key | apt-key add -
# echo "deb http://apt.vizrt.com stable main" > /etc/apt/sources.list.d/vizrt.list
# apt-get update

You can now install any packages from this repository with apt-get, e.g.: 

# apt-get install escenic-content-engine-installer

to install ece-install, or: 

# apt-get install escenic-content-engine-installer

to install the vosa scripts for managing KVM images.

There is also an unstable pool of packages in case you want to live on the bleeding edge. These packages are updated daily so beware. To live on the bleeding edge, replace stable with unstable in /etc/apt/sources.list.d/vizrt.list

If you find any issues with these packages, please report this on github: https://github.com/vizrt/ece-scripts/issues

Enjoy!
